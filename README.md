**SCHOOL MANAGEMENT SYSTEM BACKEND**
- A scalable and secure backend for a multi-role School, College, University.
- Built with Node.js, Express.js, and MongoDB, featuring role-based access control (RBAC), authentication, and modular API design for seamless integration. 
## 📌 Features

- 🔐 User Authentication (JWT)
- 📝 Create / Delete Posts
- ❤️ Like & Comment System
- 👤 User Profiles
- 📦 RESTful API
